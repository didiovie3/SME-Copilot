# **App Name**: Harbor SME Insights

## Core Features:

- Dashboard: Display key metrics and alerts in a user-friendly interface.
- Sale/Expense Logging: Enable SMEs to log daily income and expenses.
- Inventory Management: Allow SMEs to track inventory levels and costs.
- Audit Reports: Generate professional audit reports for Harbor & Co. advisors.
- Merchant List (Admin): Provide Harbor & Co. auditors with a list of all SMEs.
- Data Export (CSV): Enable auditors to export SME data in CSV format.
- Push Advice Tool (Admin): Allows admins to send notes to specific users, based on trends and reports from their stored data. It is up to the LLM tool to determine what information to present to the admin.

## Style Guidelines:

- Primary color: Deep blue (#003366) to convey professionalism and trust, in line with the existing style guidelines.
- Background color: Light blue (#D3E2F0), a desaturated version of the primary, creates a clean and calming backdrop for content.
- Accent color: Emerald green (#28A745), specified in the prompt, highlights important actions and metrics.
- Body and headline font: 'Inter', a grotesque sans-serif, combines a modern, objective, neutral look; suitable for both headlines and body text.
- Use clean and professional icons to represent different data types and actions.
- Design a mobile-first, responsive layout with clear hierarchy and intuitive navigation. Prioritize key actions with prominent "Quick Action" buttons on the homepage.
- Use subtle animations to provide feedback and enhance user experience, such as loading spinners and transition effects.
